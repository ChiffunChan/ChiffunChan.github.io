前端的必经之路——切图

  1. 获取信息

  2. 尺寸信息 -> 测量

    1. 所有数字都要测量

      1. 宽度,高度

      2. Padding,margin

      3. Border

      4. Position

      5. Text-size

      6. Line-Height

      7. 背景图位置

    2. 测量方法

      1. 放大画布 -> 矩形选框工具 -> 画一个矩形框 ->
        右侧信息面板(F8)就可以读取到宽高。

      2. 文字大小

        1. 文字图层: 看图层信息(文字属性面板可以读取)

        2. 其他: 用选框工具测量,一般选大些的字测(有误差)

  3. 颜色信息 -> 取色

  4. 切图之前

  5. 那些事需要切出来的

    1. 修饰性的(一般用在background 属性)

      1. 图标(icon),Logo

      2. 有特殊效果的按钮,文字等

      3. 非纯色的背景

  1. 内容性的(一般用在img标签)

    1. Banner(横幅标题),广告图片

    2. 文章中的一些配图

    3. 从服务器拉取过来,其实不用切图的，用img占位就可以了。

  2. 切出来的图片存为哪种类型？

    1. 修饰性的

      1. PNG24（IE6不支持半透明）

      2. PNG8 （不支持半透明）

    2. 内容性的

      1. 一般存为JPG (压缩)

点几下左边的眼睛看看就知道哪个是哪个了!!!

  1. 切图流程

  2. 隐藏文字只留背景

    1. 文字为独立图层,隐藏文字图层

      1. 找到文字图层 (用移动工具 或者 筛选)

      2. 去掉眼睛图标

->

类型筛选出所有文字类型,按住shift点击第一个,再点击最后一个,右击左边像眼睛,隐藏本图层

  1. 若文字和背景合并, 拉伸/平铺背景覆盖文字

    1. 矩形选框工具

    2. 自由变换(Ctrl+T)

    3. 使用移动工具 按Alt (+shift就能水平了)

  2. 切图 PNG24

    1. 移动工具选中所需图层(要切出来的图)(按住Ctrl多选)

    2. 右键合并图层(Ctrl+E) //单图层时不用合并

    3. 再右键复制图层到性文件(或直接拖至已有文件。(例如上一个新建的还有很多位置)
      新建Ctrl+N ) 命名为适合的名字

->

->

  1. 裁剪画布: 矩形选框 选中需要的部分 -> 图像 -> 裁剪

(或者按着Ctrl点击图层缩略图/矩形框选择 -> 裁剪工具 -> Enter键 * 2确认)

所有拖在一张后,直接多张一起导出。

  1. 切图 PNG8

带背景切 (例如一些小图标)

  1. 合并(可见)图层 (Shift+Ctrl+E) [得到整一张图]

  2. 矩形选框工具选择内容

  3. 魔棒工具去除多余部分

-从选区中减去:按住Alt

  1. 把最终选好的拖到已有文件

  2. 可平铺背景的切图

    1. 用矩形选框工具选取一块区域

    2. 复制粘贴到新文件中 (Ctrl+C -> Ctrl+N 预测剪切板,画布高度++, 背景透明
      -> Ctrl+V)

平铺内容 充满 文件的宽(x轴) 或高(y轴)。不然拉伸的时候就GG了

./media/image9.png

./media/image10.png

  1. 切片工具

适用于可以一刀切的活动页

  1. 拉参考线

  2. 选择切片工具

  3. 点击”基于参考线的切片”按钮 （就会变为切片了）

  4. 选择“切片选择工具”，双击切片重命名

  5. 保存： 文件 –> 存储为Web使用格式 (Ctrl+Alt+Shift+S) -> 缩小内容,全选切片
    -> 统一设置格式(jpg,品质- -) –> 存储。

./media/image11.png

./media/image12.png

./media/image13.png

  1. 保存

  2. 存储所需内容

    1. 复制 -> 新建 -> 粘贴 (或拖动内容至新文件)

    2. 存储为Web所用格式 (Alt+Shift+Ctrl+S)

  3. 保存哪种格式?

    1. JPG :

      1. 适合色彩丰富,没有透明度要求。

      2. 设置：品质降低到60~80左右。

    2. PNG8 :

      1. 适合色彩不太丰富（无论有无透明度要求）。

      2. 设置： 杂边 - 无 ， 扩散 - 无仿色。

      3. 如果记不住，那就把这个设置保存下来方便下一次用

    3. PNG24

      1. 适合 图片有半透明需求是

      2. 对图片无压缩，文件比较大

    4. PSD

为了保证图片质量

  1. （每个文件都）保留一份PSD，在PSD上进行修改。

  2. 修改和维护

  1. 要放更多的图片 –> 更改画布大小 (定位选在左上角)

    ./media/image15.png

  2. 移动图标

    a) 图标为独立图层 -> 使用移动工具拖动

    b) 图标为非独立图层 -> 用选区工具选中图标区域 -> 用移动工具拖动图标
    (Ctrl+X再Ctrl+V就可以把在同一图层的分开了)

  1. 裁剪掉多余画布 -> 裁剪工具

  2. 注意: 维护PNG8图片时(标题栏有”索引”) ,需更改颜色模式为RGB颜色

  3. 优化和合并

    1. Sprite（雪碧图）

把多个小图标，拼起来放在一张图，保持一定的间隔，然后使用的时候根据图片位置使用。

减少文件请求，提升网页加载速度。

  1. 压缩

    1. 无损

    2. 有损

  2. 合并 – 排列

    1. 图片之间保留空隙

    2. 优化

      1. 平衡图片的大小与质量

      2. 压缩工具

        1. 无损 Minimage （基于Node的）

        2. 有损 TinyPng （在线网站）

    3. 排列

      1. 横向排列

      2. 纵向排列

      3. 根据实际情况选择

      4. 紧密一些排列，但是不可以太乱

    4. 分类

      1. 属于一个模块的图片进行合并

      2. 大小相近的图片进行合并

      3. 色彩相近的图片合并

      4. 综合以上方式合并

    5. 合并推荐

      1. 有状态图标合并

      2. 只本页用到的图片合并

  3. 浏览器兼容方案

  1. 网页设计命名规范

一张完整的页面往往包含很多个部分，例如导航、Banner、菜单、主体、版底和Logo等，使用Photoshop设计页面时按照规定的准则命令图层或图层组，不仅有利于快速查找和修改页面效果，还可以大幅度提高切图和后期制作的工作效率。下图是网页的基本框架和结构，设计网页时，最好将同一区域的图层整合在一起。

下面是一些常用的CSS标准化设计命名，在使用Photoshop设计页面时，我们也可以按照英文名来命令每一个具体的部分。

头：header

内容：content/container

页脚：footer

导航：nav

子导航：subnav

侧栏：sidebar

栏目：column

广告条：banner

主体：main

左/右/中：left /right /center

菜单：menu

子菜单：submenu

注册：regsiter

登录：login

标志：logo

按钮：btn

图标：icon

热点：hot

新闻：news

下载：download

搜索：search

滚动条：scroll

标签页：tab

文章列表：list

小技巧：tips

标题：title

加入：joinus
